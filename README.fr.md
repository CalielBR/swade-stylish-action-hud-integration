## ✨ Fonctionnalités

- 🔗 Intégration fluide entre SWADE et Stylish Action HUD
- ⚡ HUD automatique avec toutes les actions du personnage
- 🎲 Jets 100% natifs de SWADE (Activities), sans module d'automatisation des dés supplémentaire
- 🧼 Interface claire et organisée
- 🚀 Plug and play (aucune configuration requise)
- 🧩 Compatible avec Foundry VTT V14

---

## 📦 Prérequis

- Système : SWADE (v6.0.3+)
- Modules :
  - Stylish Action HUD

---

## 🚀 Installation

1. Ouvrez Foundry VTT
2. Allez dans **Installer des modules**
3. Cliquez sur **Installer un module**
4. Collez l’URL :
   https://raw.githubusercontent.com/CalielBR/swade-StylishActionHud-integration/main/module.json

---

## 🧠 Fonctionnement

Ce module connecte :

- Le système SWADE
- Stylish Action HUD

Tous les jets (attributs, compétences, armes, pouvoirs, consommables) passent directement par l'API native de SWADE (`actor.rollAttribute`, Activities des objets), sans dépendre d'aucun module d'automatisation des dés.

👉 Résultat : un système de combat rapide, fluide et intuitif.
