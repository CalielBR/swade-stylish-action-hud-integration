## 🇬🇧 English

### 🌟 Key Features
- **Seamless Integration**: Acts as a high-performance bridge between SWADE, Stylish Action HUD, and Better Rolls 2.
- **Automated Action HUD**: Automatically populates the HUD with Attributes, Skills, Weapons, Powers, Special Actions, and Consumables.
- **Optimized for Better Rolls 2**: All actions initiated from the HUD are processed through the Better Rolls 2 engine.
- **Clean UI**: Logical categorization and wider submenus for better readability.
- **Plug and Play**: No configuration needed.
- **Foundry V13 Ready**: Fully tested for the latest version.

### 🛠️ Requirements
- **System**: Savage Worlds Adventure Edition (SWADE) (v5.1.0 or newer)
- **Modules**: Stylish Action HUD, Better Rolls for SWADE 2.
