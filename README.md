### 🌟 Key Features

-   **Seamless Integration**: Acts as a high-performance bridge between SWADE, Stylish Action HUD, and Better Rolls 2.
-   **Automated Action HUD**: Automatically populates the Stylish Action HUD with all your character's available actions, including:
    -   Attributes (Agility, Smarts, etc.)
    -   Skills (Core and custom)
    -   Weapons (Melee and Ranged)
    -   Powers
    -   Special Actions (like Grapple, Push, Wild Attack)
    -   Consumables
-   **Optimized for Better Rolls 2**: All actions initiated from the HUD are processed through Better Rolls 2, ensuring a consistent and enhanced rolling experience.
-   **Clean & Organized UI**:
    -   Simplified and clear labels for all actions.
    -   Logical categorization of actions into tabs (e.g., Combat, Powers, Skills).
    -   Wider submenu for better readability of long action lists.
-   **Plug and Play**: No configuration needed. Just install and activate the module alongside its dependencies, and it works out of the box.
-   **Foundry V13 Ready**: Built and tested for the latest Foundry VTT version.

### 🛠️ Requirements

This module requires the following to be installed and active:
-   **System**: Savage Worlds Adventure Edition (SWADE) (v5.1.0 or newer)
-   **Module**: Stylish Action HUD
-   **Module**: Better Rolls for SWADE 2

### 🚀 Installation

1.  In Foundry VTT, go to the **Add-on Modules** tab in the Configuration and Setup screen.
2.  Click **Install Module**.
3.  In the "Manifest URL" field, paste the following link:
    ```
    https://raw.githubusercontent.com/CalielBR/swade-StylishActionHud-integration/main/module.json
    ```
4.  Click **Install** and wait for the installation to complete.
5.  Activate this module and all required dependencies in your game world's module settings.

## 🇧🇷 Português

### 🌟 Principais Funcionalidades

-   **Integração Perfeita**: Atua como uma ponte de alta performance entre o sistema SWADE, o Stylish Action HUD e o Better Rolls 2.
-   **HUD de Ações Automatizado**: Preenche automaticamente o Stylish Action HUD com todas as ações disponíveis do seu personagem, incluindo:
    -   Atributos (Agilidade, Astúcia, etc.)
    -   Perícias (Básicas e customizadas)
    -   Armas (Corpo a corpo e à distância)
    -   Poderes
    -   Ações Especiais (como Agarrar, Empurrar, Ataque Feroz)
    -   Consumíveis
-   **Otimizado para Better Rolls 2**: Todas as ações iniciadas pelo HUD são processadas através do Better Rolls 2, garantindo uma experiência de rolagem consistente e aprimorada.
-   **Interface Limpa e Organizada**:
    -   Rótulos simplificados e claros para todas as ações.
    -   Categorização lógica de ações em abas (ex: Combate, Poderes, Perícias).
    -   Submenu mais largo para melhor legibilidade de listas de ações longas.
-   **Instale e Use (Plug and Play)**: Nenhuma configuração é necessária. Apenas instale e ative o módulo junto com suas dependências, e ele funcionará imediatamente.
-   **Pronto para Foundry V13**: Construído e testado para a versão mais recente do Foundry VTT.

### 🛠️ Requisitos

Este módulo requer que os seguintes pacotes estejam instalados e ativos:
-   **Sistema**: Savage Worlds Adventure Edition (SWADE) (v5.1.0 ou mais recente)
-   **Módulo**: Stylish Action HUD
-   **Módulo**: Better Rolls for SWADE 2

### 🚀 Instalação

1.  No Foundry VTT, vá para a aba **Add-on Modules** na tela de Configuração.
2.  Clique em **Install Module**.
3.  No campo "Manifest URL", cole o seguinte link:
    ```
    https://raw.githubusercontent.com/CalielBR/swade-StylishActionHud-integration/main/module.json
    ```
4.  Clique em **Install** e aguarde a conclusão da instalação.
5.  Ative este módulo e todas as suas dependências nas configurações de módulos do seu mundo de jogo.
